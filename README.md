# Анализатор страниц
Ссылка на домен: https://python-project-83-ad5g.onrender.com

Анализатор страниц — это Python-приложение для анализа веб-страниц.
Сервис предназначен для проверки страниц на SEO-пригодность.

Установка

Клонируйте репозиторий:
git clone
cd python-project-83

Установите зависимости:
make install

Запуск приложения:
make start
